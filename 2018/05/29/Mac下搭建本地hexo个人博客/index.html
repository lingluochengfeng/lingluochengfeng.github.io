本地hexo博客搭建

本文介绍了从零开始搭建hexo个人博客的过程

内容简介

  • 安装git
  • 安装node.js
  • 安装hexo
  • 创建本地服务，启动Hello World

安装git

进入官网自行下载安装git，这里附上git官方网址：git官网下载
安装完成后我们需要查看git是否安装成功
打开终端，输入以下命令查看是否安装成功

如果看到了git返回的信息，则表示git安装成功

安装node.js

进入[node.js官网]自行进行下载和安装
安装成功后，打开终端，使用以下命令查看node.js是否安装成功

如果看到了node.js的版本信息，则node.js安装成功

安装hexo

进行这一步之前，必须保证git和node.js已经安装完成
我们可以通过一条命令安装hexo

如果没有报错信息，等待其安装完成即可。如果出现报错信息，一般是权限问题，只要在那条指令前加上

$ hexo init # hexo会在目标文件夹建立网站所需要的所有文件

$ sudo npm install hexo-server

$ hexo g
$ hexo server

启动服务后，打开浏览器，按照终端的输出信息输入地址，一般是```http://localhost:4000

如果没什么问题的话，就能访问到hexo的初始化页面(即Hello World页面),结束在终端中用组合件:control+C

建议以后在将博文正式上传到GitHub之前先在本地预览，确定没问题，再上传。

到此为止，本地搭建hexo的步骤就已经完成了，欢迎列位看官批评指正
